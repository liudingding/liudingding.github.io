Mac 下使用Hexo + Github 搭建自己的blog

  • Github
  • Hexo
  • Mac

前言

一直想用blog来记录自己的学习路径,总是嫌麻烦,现在终于下定决心把blog建了起来,就从这一篇开始进阶之路吧!

第一步 Github和配置SSH

在搭建之前你需要一个github账号。如果没有账号，请注册；然后你要创建一个仓库New repository，名称为你github用户名.github.io。

先生成SSH KEY

$ssh-keygen -t rsa -C "xxxx@xxxx.com” #引号里为你的邮箱地址

登陆github页面 点击自己的头像->Setting->SSH Public Keys->Add
把生成的key(id_rsa.pub)内容copy到输入框

测试是否连接

$ssh git@github.com

设置个人信息

$git config --global user.name "xxx”
$git config --global user.email xxxx@gmail.com

第二步 Homebrew安装

终端运行

$ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

第三步 git

如果你已经安装了xcode,这一步可以跳过,否则终端运行

$brew install git

第四步 node.js

终端运行

$brew install node

第五步 Hexo安装

任意地方创建一个文件夹,终端进入文件夹然后运行

$nmp install -g hexo 
$hexo init 
$npm install

这个时候你就可以使用一下命令查看默认页面

$hexo generate #创建页面  简写 hexo g
$hexo server  #启动服务器  简写 hexo s

出现

Hexo is running at http://0.0.0.0:4000/

证明服务器已经启动(可以浏览器上查看这个页面) ctrl+c退出

修改hexo目录下_config.yml deplo下的三行需要缩进，否者无法识别。冒号后面必须空一格,不然也没法识别

deploy:
    type: git
    repo: git@github.com:xxxxxx/xxxxxx.github.io.git
    branch: master

执行

hexo deploy   #部署到github上

如果报错

hexo ERROR Deployer not found: git

执行

npm install hexo-deployer-git —save

P.S

hexo常用指令

hexo new "postName" #新建文章
hexo new page "pageName" #新建頁面
hexo generate #生成靜態頁面至public目錄
hexo server #開啟預覽訪問端口（默认端口4000，'ctrl + c'關閉server）
hexo deploy #將.deploy目錄部署到GitHub

简写

hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy

结束

新建文章时

hexo new "postName" #新建文章

在source文件下_post里面会有一个.md文件,你可以使用Markdown来写你的博客写完之后

hexo g #生成页面
hexo d $发布就好啦
